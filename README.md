# Week 5 Spark Assignment

## Objective

To understand Spark fundamentals and perform data cleaning, transformation, filtering, aggregation, and schema modification using Spark DataFrames.

## Tools Used

- Databricks
- Apache Spark
- PySpark

## Dataset

Telco Customer Churn Dataset

## Tasks Performed

- Loaded dataset into Spark DataFrame
- Removed duplicates
- Handled null values
- Applied filtering conditions
- Performed aggregations
- Used groupBy transformations
- Modified schema
- Built a complete Spark processing pipeline
